# Example Datasets

This directory contains small datasets meant for local demo runs.

## Included files

- `sample_datasets/bike_demand_regression_demo.csv`
  - simple regression-oriented dataset for validating the experiment flow
- `sample_datasets/customer_churn_classification_demo.csv`
  - simple classification-oriented dataset for validating model selection and ranking

These files are intentionally small and easy to inspect. They are here to support local setup, portfolio demos, and smoke testing of the main workflow.
